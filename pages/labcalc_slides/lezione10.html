# Array * Esercizio: scrivete un programma che calcoli la media di una sequenza di numeri data * Svolgimento: in C non sappiamo ancora come memorizzare una sequenza!
* Definire una variabile per ogni elemento non è pratico (`int elem1, elem2, ...`) * Le strutture dati che ovviano a questo problema si chiamano *array* * Un array è un insieme di elementi **dello stesso tipo** * Possiamo pensarlo come una serie di caselle contigue * Ad ogni casella è associato un indice intero e non negativo (cioè $\geq 0$) * Un esempio di array di interi di $N$ caselle: |Indice|0|1|2|3|...|$N - 1$| |-|-|-|-|-|-|-| |Contenuto |0 | -2| 42| 1|...| -5|
Alla prima casella di memoria è associato l'indice 0, all'ultima l'indice $N - 1$
--- # Esempio di `array` - media di una sequenza ```C [|1|5-7] double sequenza[7] = {0., 5.2, -1.1, 4.7, 8.0, 12.1, -3.2}; double media; int i; for(i = 0, media = 0.; i < 7; i++) { media += sequenza[i]; } media /= 7.0; printf("La media è %lf\n", media); ``` * La sintassi per definire un array è `tipo nome[lunghezza];` * Gli elementi possono essere inizializzati mettendo i valori separati da virgole tra `{}` * Si accede all'elemento $i$-esimo tramite la sintassi `nome[i]` * `i` deve essere un numero intero non negativo * `i` deve essere $\leq N - 1$ * Esempio: `sequenza[3]` è il *quarto* elemento dell'array, che in questo caso vale `4.7` --- # Cosa succede "sotto al cofano" * `double sequenza[7]` chiede al compilatore di allocare in memoria sette `double` * I sette `double` (64 bit ognuno) vengono allocati uno vicino all'altro * Il compilatore associa al nome `sequenza` *l'indirizzo di memoria* della prima casella * Usando `sequenza[i]` si accede al valore contenuto nella $i$-esima cella di memoria a partire dalla prima
Il programma non controlla se l'$i$-esima cella fa effettivamente parte dell'array
* La memoria che precede e segue `sequenza` può contenere qualsiasi cosa: finire *out of bounds* significa leggere o scrivere in zone di memoria non di pertinenza dell'array
|||`sequenza[0]`|`sequenza[1]`|`sequenza[2]`|`sequenza[3]`|`sequenza[4]`|`sequenza[5]`|`sequenza[6]`||| |-|-|-|-|-|-|-|-|-|-|-| |...|...|0.|5.2|-1.1|4.7|8.0|12.1|-3.2|...|...|
--- # Indicizzare gli array
|||`sequenza[0]`|`sequenza[1]`|`sequenza[2]`|`sequenza[3]`|`sequenza[4]`|`sequenza[5]`|`sequenza[6]`||| |-|-|-|-|-|-|-|-|-|-|-| |...|...|0.|5.2|-1.1|4.7|8.0|12.1|-3.2|...|...|
  • sequenza[3]
  • sequenza[8 - 2]
  • sequenza[idx] con int idx = 4
  • sequenza[idx - 4]
  • sequenza[-2] $\to$ undefined behaviour
  • sequenza[7] $\to$ undefined behaviour
  • sequenza[x] con double x = 4 $\to$ l'indice deve essere intero!

--- # La dimensione degli array * La dimensione (o lunghezza) degli array deve essere nota a *compile time* * È comune che la dimensione di un array venga usata più volte in un codice * Per inizializzare più di un array * Nei cicli `for` che iterano sugli elementi degli array * In questi casi l'opzione migliore è utilizzare una macro
```C #define DIM_ARRAY 10 int main() { double seq[DIM_ARRAY]; int counter[DIM_ARRAY]; // altre operazioni int i; for(i = 0; i < DIM_ARRAY; i++) { // opera sugli array } return 0; } ```
* Cambiare la macro cambia la dimensione ovunque nel codice! * Migliore leggibilità * Non si confonde con altri valori
--- # Inizializzare gli array ```C [1-3|5-8|10|12|14-19] double numeri[3] = {-4.2, 0.0, 2.1}; // valido solo al momento della dichiarazione numeri[3] = {-4.2, 0.0, 2.1}; // ERRORE DI COMPILAZIONE numeri = {-4.2, 0.0, 2.1}; // ERRORE DI COMPILAZIONE double numeri_2[3]; // dichiarazione e inizializzazione separati numeri_2[0] = -4.2; numeri_2[1] = 0.0; numeri_2[2] = 2.1; double numeri_3[3] = {0.0}; // con questa sintassi *tutti* gli elementi varranno 0 double numeri_4[3] = {1.0}; // ATTENZIONE, numeri[0] == 1.0, // ma numeri[1] == numeri[2] == 0.0 double numeri_5[30]; int i; for(i = 0; i < 30; i++) { // inizializzazione tramite un ciclo for numeri_5[i] = 2 * (i + 1); printf("L'elemento %d-esimo ha valore %lf\n", i, numeri_5[i]); } ```
Assicuratevi sempre (ad es. con dei printf) che gli array siano inizializzati bene!
--- # Esempio: inizializzare da riga di comando * Il programma chiede all'utente di inserire una sequenza di numeri, che vengono ristampati sottraendone la media ```C [|2,5|9|10-13|14|16-18] #include #define DIM 5 int main() { double numeri[DIM]; // senza inizializzazione perché leggiamo i valori dall'input int i; printf("Inserire %d numeri: \n", DIM); double media = 0.; for(i = 0; i < DIM; i++) { scanf("%lf", &numeri[i]); // come vedremo, "&numeri[i]" è equivalente a "numeri + i" media += numeri[i]; } media /= DIM; for(i = 0; i < DIM; i++) { printf("%d %lf\n", i, numeri[i] - media); } return 0; } ``` --- # Esempio: inizializzare da riga di comando * Il programma chiede all'utente di inserire una sequenza di numeri, che vengono ristampati sottraendone la media ```text $ ./es_input_array Inserire 5 numeri: 5.2 -9.3 0 4.5 1.2 0 4.880000 1 -9.620000 2 -0.320000 3 4.180000 4 0.880000 ```
